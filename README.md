# Thrillok M - AI Research Engineer Portfolio

Scaffolded Next.js + TypeScript portfolio optimized for personal branding.

Quick start:

1. Install dependencies

   npm install

2. Run dev server

   npm run dev

Build for production:

   npm run build
   npm start

Export static for GitHub Pages:

   npm run export

Deploy to Vercel: connect repository and use default settings.
